создать на .NET 4.5 (WPF) таб-контрол (аналогичный контролу в браузерах) с учетом следующих условий: 
- вкладки расположены сверху; 
- на каждой вкладке нужны надпись и кнопка "Close" по которой вкладка закрывается;
- по закрытию вкладка и содержимое должны выгружаться из памяти, проверяться будет ANTS Memory Profiler 8;
- вкладки должны менять размер при изменении ширины окна и изменении количества вкладок; 
- надо обеспечить возможность менять порядок вкладок с помощью dran&drop.
